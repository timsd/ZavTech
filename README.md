# ZavTech

![ZavTech Logo](public/favicon-32x32.png)

**Professional Technical Solutions | Web & Mobile & Arduino Development**

ZavTech provides professional technical solutions including web development, mobile app development, Arduino programming, and renewable energy systems for businesses in Nigeria and globally.

## 🚀 Features

- **Modern React Application** - Built with React 18, TypeScript, and Vite
- **Responsive Design** - Mobile-first approach with Tailwind CSS
- **Component Library** - Powered by Radix UI and shadcn/ui components
- **Professional UI/UX** - Clean, modern interface with smooth animations
- **SEO Optimized** - Meta tags, Open Graph, and Twitter Card support
- **Performance Focused** - Fast loading with optimized assets

## 🛠️ Tech Stack

- **Frontend Framework:** React 18 with TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **UI Components:** Radix UI + shadcn/ui
- **Icons:** Lucide React
- **Fonts:** Inter & Montserrat (Google Fonts)
- **State Management:** React Query (TanStack Query)
- **Form Handling:** React Hook Form with Zod validation
- **Theme:** Next Themes for dark/light mode support

## 📦 Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/zavtech.git
   cd zavtech
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   # or
   bun install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:5173`
